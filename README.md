## What Is NEURAMED?

NEURAMED is a **full-stack AI diagnostic platform** that puts three specialized clinical intelligence agents вҖ” Voice Diagnosis, Medical Imaging AI, and OCR Report Reader вҖ” into a single, unified dashboard. Built for speed, accuracy, and clinical precision.

Every diagnosis is powered by **LLaMA 3 70B via Groq** (sub-second inference), persisted to a structured database, broadcast in real-time via WebSocket, and exportable as a clinical PDF вҖ” all from a dark, instrument-grade UI that looks like it belongs in an ICU.

This is not a prototype. This is a production-grade system.

## The Three Agents

<br />

### рҹҺӨ Voice Diagnosis Agent
Accepts spoken or typed symptoms. Transcribes audio via **ElevenLabs Scribe**, processes the transcript through a medical-grade LLaMA 3 prompt, and returns a structured differential diagnosis with confidence scoring, urgency classification, and ranked treatment recommendations вҖ” in under 2 seconds.

```
Input:  "chest pain radiating to left arm, sweating, shortness of breath"
Output: Acute Coronary Syndrome (94%) В· Aortic Dissection (61%) В· GERD (23%)
        Urgency: CRITICAL  |  Confidence: 0.94  |  Processing: 1.2s
```

<br />

### рҹ§  Medical Imaging AI Agent
Accepts CT scans, MRIs, X-Rays, and Ultrasound images. Runs a full **OpenCV pipeline** вҖ” CLAHE histogram equalization, Gaussian denoising, Otsu thresholding, contour analysis вҖ” to detect and annotate anomaly regions. LLaMA 3 then interprets the region statistics into clinical radiological findings. Returns the original scan side-by-side with a fully annotated version.

```
Input:  CT scan JPEG/PNG/DICOM
Output: Annotated image + bounding boxes + confidence per region
        + LLM radiological impression + follow-up recommendations
```

<br />

### рҹ“„ OCR Report Reader Agent
Accepts PDF medical reports or scanned document images. **PyTesseract** extracts raw text, a regex-based section parser structures it into clinical sections (Chief Complaint, Findings, Impression, Plan), and LLaMA 3 generates a 3-sentence summary plus key findings, abnormal flags, and medication list.

```
Input:  Any medical report PDF or image
Output: Structured sections + AI summary + abnormal flags + medications
        + urgency classification + exportable findings
```

- **Real-Time Dashboard** вҖ” WebSocket-powered live feed, no page refresh needed
- **Three AI Agents** вҖ” Each independently callable, testable, and deployable
- **Groq Inference** вҖ” LLaMA 3 70B at sub-2-second response times
